Appearance Index API

Finding all appearances of an address is known to be hard on Ethereum. For example, balances can change without transfer and not all event parameters are indexed.

At TrueBlocks we do our best to pick every address appearance out there. Key provides a single, simple API that does one thing well: it returns complete list of appearances so you know which transactions to fetch. It has been proven by companies and data scientists to return significantly more than other indexers.
